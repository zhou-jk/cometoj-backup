C0344 [USACO]联系

内存限制：256 MB 时间限制：1000 ms

题目描述

奶牛们开始对用射电望远镜扫描牧场外的宇宙感兴趣。最近，他们注意到了一种非常奇怪的脉冲调制微波从星系的中央发射出来。他们希望知道电波是否是被某些地外生命发射出来的，还是仅仅是普通的的星星发出的。

帮助奶牛们用一个能够分析他们在文件中记下的记录的工具来找到真相。他们在寻找长度在 $A$ 到 $B$ 之间（含）在每天的数据文件中重复得最多的比特序列 $(1 \le A \le B \le 12)$。他们在找那些重复得最多的比特序列。一个输入限制告诉你应输出多少频率最多的序列。

符合的序列可能会重叠，并且至少重复一次的序列会被计数。

输入格式

第一行： 三个用空格分隔的整数：$A, B, N$；$(1 \le N < 50)$

第二行及以后：一个最多 $200,000$ 字符的序列，全是 $0$ 或 $1$；每行字符数不大于 $80$。

输出

输出 $N$ 个频率最高的序列（按照频率由高到低的次序）。由短到长排列频率相同的这些序列，如果长短相同，按二进制大小排列。如果出现的序列个数小于 $N$，输出存在的序列。

对于每个存在的频率，先输出单独包含该频率的一行，再输出以空格分隔的这些频率。每行六个（除非少于六个剩下）。
